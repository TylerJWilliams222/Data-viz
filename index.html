About This Project

This interactive map visualizes vending machine locations across the Royal Holloway University of London campus. Each marker represents a vending machine location and can be clicked to view practical access information and a building plaque image for easier identification.

The project highlights how small campus facilities can become important accessibility resources. For students who need quick access to sugary food or drinks, including those managing blood sugar levels, knowing the nearest vending machine can make campus navigation more reliable and less stressful.

Rather than presenting the data as a static list, the map turns these locations into a visual and interactive campus guide, making it easier for all students to discover and access these resources when needed.
